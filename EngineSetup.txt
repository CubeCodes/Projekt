--------------LINUX------------------
1. Jdete na stranku
	https://www.unrealengine.com/en-US/
2. Vytvorte si ucet Epic games (je potreba se prihlasit pro stazeni UE4)
3. Bezte do nastaveni uctu → propojene ucty (connected accounts)
4. Pripojte svůj Github ucet 
5. Do terminalu zadejte prikaz
	git clone https://github.com/EpicGames/UnrealEngine-(branch_name).git
6. Spustit engine a prihlasit se
7. Najdete slozku s projektem (mela by se zobrazit automaticky) a otevrite ji


--------------WINDOWS-------------------
1. Jdete na stranku
	https://www.unrealengine.com/en-US/
2. Prihlaste se/vytvorte ucte Epic games (je potreba se prihlasit pro stazeni UE4)
3. Stahnete si Epic games launcher a prihlaste se
4. V zalozce Unreal Engine naleznete tlacitko download s verzi enginu, kterou chcete stahnout
7. Najdete slozku s projektem (mela by se zobrazit automaticky) a otevrite ji
